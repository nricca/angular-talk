Filtres

  • Traitement ou filtrage des données
  • Simplification syntaxique
  • Différents filtres pré-inclues
  • Possibilité de créer ses propres filtres

Exemple

Input : <input type="text" ng-model="monPrix">
Value : {{ monPrix | currency:"CHF "  }}

Input : 
Value : {{ monPrix | currency:"CHF " }}
