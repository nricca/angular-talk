Contrôleur

  • Instancié via l'attribut ng-controller
  • Lié à une portion de vue via la variable $scope
  • Contient la logique métier d'une vue

Exemple

<div ng-controller="MyCtrl">
    Prénom : <type="text" ng-model="name">
    <button ng-click="hello()">Salutations</button>
</div>
                
angular.module("app").controller("MyCtrl", function($scope) {

    $scope.name = "Batman";

    $scope.hello = function () {
        alert("Salut " + $scope.name);
    }
});
